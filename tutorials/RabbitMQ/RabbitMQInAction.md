# 一、安装RabbitMQ

## 1、下载

下载地址获取页面： http://www.rabbitmq.com/download.html

docker镜像页面： https://hub.docker.com/_/rabbitmq/

## 2、安装

### 2.1、docker安装

- 安装镜像

```shell
[emon@emon ~]$ docker run -d --hostname my-rabbit -p 5672:5672 -p 15672:15672 rabbitmq:3.7.7-management
```

- 验证

http://IP:15672
